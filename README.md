• Designed and developed a console-based Railway Ticket Booking System using Core Java, enabling
users to book, cancel, and view train tickets through a menu-driven interface.
• Applied Object-Oriented Programming (OOP) principles such as classes, objects, encapsulation, and
constructors to create a modular and maintainable application structure.
• Implemented seat management using arrays, where each index represents a seat and stores
passenger details dynamically at runtime.
• Developed a Train module to handle all major operations including seat booking, seat cancellation,
seat availability checking, and displaying booked tickets.
• Built a custom exception to handle cases such as invalid seat numbers, already booked seats, and
cancellation of empty seats, improving system reliability..
